# FIGUS

Aplicación Next.js para cargar e intercambiar figuritas del Mundial 2026.

## Desarrollo local

```bash
npm install
npm run dev
```

Abrí [http://localhost:3000](http://localhost:3000).

## Verificaciones

```bash
npm run lint
npm run build
```

## Despliegue

El proyecto está listo para importarse y desplegarse en Vercel sin configuración adicional.
